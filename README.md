# 📧 InboxFlow – Django Email Sending Web App

🚀 **Live Demo:**
👉 https://priashapatle.pythonanywhere.com

---

## 📌 Project Overview

InboxFlow is a Django-based web application that allows users to send emails directly from a web interface.
It also maintains email history and provides a simple dashboard for managing contacts.

---

## ✨ Features

* 📤 Send emails using Gmail SMTP
* 📜 Email history tracking
* 👤 User authentication (Login/Register/Logout)
* 📇 Contact management
* 📊 Dashboard view

---

## 🛠️ Tech Stack

* **Backend:** Django (Python)
* **Frontend:** HTML, CSS, JavaScript
* **Database:** SQLite
* **Deployment:** PythonAnywhere

---

## ⚙️ Setup Instructions

### 1️⃣ Clone the repository

```bash
git clone https://github.com/priashapatle/inboxflow.git
cd inboxflow
```

### 2️⃣ Create virtual environment

```bash
python -m venv env
env\Scripts\activate   # Windows
```

### 3️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the server

```bash
python manage.py runserver
```

---

## 🔐 Environment Variables

Make sure to configure:

* `SECRET_KEY`
* `DEBUG`
* `EMAIL_HOST_USER`
* `EMAIL_HOST_PASSWORD` (Gmail App Password)
